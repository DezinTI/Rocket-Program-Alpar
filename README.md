## 🎯 Projetos em Destaque

### 1. **DzDex (Whateverdex)**
- **Descrição**: Sistema de gerenciamento de cards com imagens, vídeos do YouTube e categorias
- **Tecnologias**: HTML5, CSS3, JavaScript ES6+, LocalStorage
- **Funcionalidades**: CRUD completo, filtros, modal interativo, categorias dinâmicas
- **Acesso**: [Ver DzDex](https://dzdex-dezinti.herokuapp.com/)
- **Documentação**: [DzDex-README.md](./DzDex-README.md)

### 2. **Café Tatuquas**
- **Descrição**: Site completo para cafeteria com menu, horários e formulário de contato
- **Funcionalidades**: Cardápio organizado, tabela de horários, formulário responsivo

### 3. **DezinTI**
- **Descrição**: Site institucional com layout baseado em tabelas
- **Funcionalidades**: Apresentação da empresa, formulário de orçamento

### 4. **DevCast Podcast**
- **Descrição**: Site de podcast com vídeos e áudios
- **Funcionalidades**: Reprodução de mídia, episódios organizados

### 5. **Cartão de Perfil**
- **Descrição**: Perfil profissional com links para redes sociais
- **Funcionalidades**: Design moderno, links para GitHub/LinkedIn, foto personalizada

## 🛠️ Como executar

### Opção 1: Abrir diretamente
1. Abra `index.html` diretamente no navegador
2. Navegue pelos projetos através dos links

### Opção 2: Servidor local (recomendado)
1. Use a extensão **Live Server** no VS Code
2. Ou execute um servidor local:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# Acesse: http://localhost:8000
```

### Opção 3: GitHub Pages
- **Portfólio online**: [https://dezinTI.github.io/Rocket-Program-Alpar/](https://dezinTI.github.io/Rocket-Program-Alpar/)
- Todos os projetos funcionam online
