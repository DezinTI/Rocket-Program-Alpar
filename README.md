# Declaração de Váriaveis

- VAR Cria váriaveis com dados que podem ser alterados

- LET Tambem cria váriaveis com dados que podem ser alterados, mas obedece melhor aos escopos

- CONST Cria váriaveis cujo o valor não pode ser alterado, como o PI(3,14)
